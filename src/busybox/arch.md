# arch

```sh
$ arch
```

Print system architecture

- 打印系统的架构

## Example

```sh
lab:~$ arch
x86_64
```

## Architectures

不同架构有不同的系统镜像，运行软件所用的安装包也不同。如 [_Alpine Linux downloads_](https://alpinelinux.org/downloads/) 有为不同架构准备的系统镜像，个人电脑常见的架构有：

- x86_64 / amd64
- aarch64 / arm64
